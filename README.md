# RLE_BMP_Image_Viewer
RLE로 압축된 BMP 파일 뷰어

RLE로 압축된 BMP 파일이나, BMP 파일을 읽어들여서 영상처리 및 뷰어 기능을 수행하는 프로그램 입니다.

사용언어 : C++
사용 툴 : MFC
